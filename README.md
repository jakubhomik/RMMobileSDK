# RMMobileSpec





## Installation

RMMobileSpec is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:


pod 'RMAppFramework', :git => 'https://github.com/RMTelephone/RMAppFramework.git'
